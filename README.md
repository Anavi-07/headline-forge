# 📰 Headline Generator
A fun Python project that generates random fake news headlines.

## 🚀 Features
- Category-based headline generation (Funny, Tech, Famous)
- Random and creative outputs
- Interactive CLI program

## 🛠️ Tech Used
- Python
- Random module

## ▶️ How to Run
1. Install Python
2. Run the file:
   python main.py
